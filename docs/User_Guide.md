# User Guide

# JanDhan Analytics AI

**Version:** 1.0

---

# 1. Introduction

Welcome to **JanDhan Analytics AI**, an AI-powered platform designed to improve transparency in public spending by transforming government budget and expenditure data into interactive dashboards, intelligent insights, and predictive analytics.

This guide explains how to use the platform effectively.

---

# 2. System Requirements

Before using the platform, ensure the following requirements are met:

### Supported Browsers

* Google Chrome (Recommended)
* Microsoft Edge
* Mozilla Firefox
* Safari

### Internet Connection

A stable internet connection is required.

### Screen Resolution

Recommended:

* 1366 × 768 or higher

# 4. Login

Step 1

Open the application.

Step 2

Enter your registered email address.

Step 3

Enter your password.

Step 4

Click **Login**.

After successful authentication, you will be redirected to the Dashboard.

---

# 5. Dashboard

The Dashboard provides a complete overview of public spending.

Available widgets include:

* Total Budget Allocation
* Total Expenditure
* Budget Utilization
* Number of Departments
* Number of Districts
* Government Schemes
* Recent AI Alerts

Users can filter information by:

* Financial Year
* Department
* District
* Government Scheme

---

# 6. Analytics Module

The Analytics page provides detailed expenditure analysis.

Available analyses include:

* Department-wise Spending
* District-wise Spending
* Scheme-wise Analysis
* Year-wise Comparison
* Monthly Trends
* Budget Utilization

Interactive charts allow users to zoom, filter, and compare data.

---

# 7. AI Assistant

The AI Assistant enables users to interact with the platform using natural language.

Example Questions

* Which department spent the highest budget?
* Show education spending in 2025.
* Which district underutilized its allocated budget?
* Compare health and education expenditure.
* Predict next year's spending.

The assistant provides concise answers, explanations, and recommendations based on the available dataset.

---

# 8. Upload Dataset (Administrator)

Administrators can upload new government expenditure datasets.

Supported Formats

* CSV
* Excel (.xlsx)

Upload Process

1. Navigate to **Upload Dataset**.
2. Select a CSV or Excel file.
3. Click **Upload**.
4. The system validates the data.
5. If validation succeeds, the data is stored in the database.
6. The dashboard is automatically updated.

---

# 9. AI Insights

The platform automatically generates summaries such as:

* Department Performance
* Budget Utilization
* Spending Trends
* Significant Changes
* Potential Anomalies

These insights help users quickly understand the data without manually analyzing large spreadsheets.

---

# 10. Budget Forecasting

The Forecasting module predicts future expenditure using historical data.

Users can:

* Select a department.
* Choose a financial year.
* Generate spending predictions.
* View confidence scores.

Forecast results should be interpreted as analytical estimates rather than official financial projections.

---

# 11. Anomaly Detection

The platform identifies unusual spending behavior.

Examples include:

* Overspending
* Underutilized Budgets
* Sudden Spending Spikes
* Missing or Incomplete Records

Alerts are displayed on the dashboard and can be reviewed in detail.

# 16. Frequently Asked Questions (FAQs)

### Q1. Can I upload my own dataset?

Yes. Administrators can upload CSV or Excel datasets.

---

### Q2. Is the forecast always accurate?

No. Forecasts are generated using historical data and should be considered predictive estimates.

---

### Q3. Can I download reports?

Yes. Reports are available in PDF and CSV formats.

---

### Q4. Does the AI modify government data?

No. The platform analyzes data but does not modify official government records.

---

### Q5. Can I compare multiple departments?

Yes. The Analytics module supports department-wise and district-wise comparisons.

---

# 17. Troubleshooting

| Problem                  | Possible Solution                              |
| ------------------------ | ---------------------------------------------- |
| Unable to Login          | Verify email and password                      |
| Dashboard Not Loading    | Check internet connection and refresh the page |
| Dataset Upload Failed    | Verify file format and required columns        |
| AI Response Delayed      | Retry after a few moments                      |
| Report Generation Failed | Ensure data has been successfully processed    |
