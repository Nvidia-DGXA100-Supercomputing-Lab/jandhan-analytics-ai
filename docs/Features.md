# JanDhan Analytics AI

---

# Overview

JanDhan Analytics AI is an AI-powered public spending analytics platform designed to improve transparency, accountability, and accessibility of government financial data. The platform combines interactive dashboards, Artificial Intelligence, Machine Learning, and predictive analytics to help citizens and organizations understand how public funds are allocated and utilized.

---

# Core Features

## Interactive Dashboard

The dashboard provides a centralized view of government spending through modern visualizations.

Features include:

* Total Budget Allocation
* Total Expenditure
* Budget Utilization Percentage
* Department-wise Spending
* District-wise Spending
* Government Scheme Overview
* Financial Year Comparison
* KPI Cards
* Interactive Charts
